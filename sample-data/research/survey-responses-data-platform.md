## Open-Ended Responses — Key Themes

### "What's the #1 thing you'd fix?" (Top responses by frequency)
1. Reference data consistency (14 mentions)
2. Add real-time corporate actions (9 mentions)
3. Python API / better developer tools (7 mentions)
4. Data catalog so I know what's available (6 mentions)
5. Faster data access provisioning (5 mentions)
6. Historical data quality / point-in-time (4 mentions)

### Selected Verbatims

**Trading:**
> "I trust Bloomberg more than our own system. That's a problem."

> "If I could get one consolidated view of my positions with live prices and pending corporate actions, I'd save an hour every morning."

**Operations:**
> "We need to stop manually processing corporate actions. Every other firm our size has automated this."

> "The settlement breaks from bad reference data cost us real money. This should be priority #1."

**Quant:**
> "Give me a Python SDK and a data catalog and I'll stop complaining. That's all I need."

> "Point-in-time data is table stakes for any serious quant platform. We're behind."

**Compliance:**
> "When the regulator asks how we processed a corporate action, I need a clear audit trail. Right now I have to manually reconstruct it from emails and spreadsheets."
